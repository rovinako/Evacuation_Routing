# Evacuation_Routing

Evacuation routing  is a web-based evacuation routing system that uses graph algorithms to find optimal paths in a crisis scenario.

The system models locations as nodes and roads as weighted edges with factors like danger, travel time, congestion, and road condition.

We implemented:
- Dijkstra’s Algorithm (main) to find the safest path
- A* Algorithm for comparison using heuristic-based search

The application allows users to select a start and destination and computes the best route based on different modes such as safest, fastest, and balanced.

This project demonstrates how algorithms can support decision-making under uncertainty and limited resources.
